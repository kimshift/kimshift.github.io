## 对接微信公众号准备

- 注册微信公众号-订阅号(凭个人身份即可注册)
- 注册小程序测试账号
- 下载安装微信开发者工具
- 模拟公网 IP 端口方式
  - 申请云服务器
  - 利用公司网络的公网 IP 进行映射
  - 使用内网穿透工具，例如 sunny-ngrok、nps 等等

## 微信扫一扫步骤

1. 使用内网穿透技术:
2. 公众号上配置相关信息
   - 获取公众号的 Appid/selectKey
   - 配置白名单
   - 配置可信域名-下载文件到项目的根目录
3. 调用 api 初始化微信环境
4. 调用 api 唤醒微信扫一扫

5. 处理扫码成功回调
