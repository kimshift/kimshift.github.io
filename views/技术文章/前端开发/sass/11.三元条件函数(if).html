    11.三元条件函数(if)

    11.三元条件函数(if)

    琴时 2024-04-24 16:12:01 css sass

    # 如何使用

    if($condition,$if-true,$if-false);
    1

    判断$condition，如果条件成立，则返回$if-true 的结果，如果条件不成立，则返回$if-false 的结果。

    案例：

    if 的写法（浅色与深色模式）

    $theme: "light";
.container {
  @if $theme== "light" {
    color: #000;
  } @else {
    color: #fff;
  }
}
